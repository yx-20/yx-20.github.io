2D Game

Introduction

This project implements a 2D game in C++. Player controls the skull to attack the pumpkins in the night. WASD controls movement, SPACE controls long-distance attack, F controls short-distance attack. The skull will be killed by pumpkins when it collided with pumpkins in the walk state, in other words when no sword in hand.
